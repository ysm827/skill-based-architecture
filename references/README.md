# References

Topic map lives in root [`REFERENCE.md`](../REFERENCE.md) — both task-oriented and by-topic indexes are there. New references should be added to the by-topic table in `REFERENCE.md` and linked directly from their callers (no two-layer stub).

## Why this directory has its own README

`references/` is a flat collection of topic files. Maintainers occasionally land here directly and need a one-line orientation. The full table is in `REFERENCE.md` to avoid two indexes drifting against each other.
